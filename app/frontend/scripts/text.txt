Шаг 1.

Работа с книгами
		- Пользователь добавляет книгу на сайт.
		- Посетитель ищет книгу с помощью поиска.
		- Пользователь ставит книге оценку.
		- Пользователь удаляет книге оценку.
		- Пользователь изменяет книге оценку.
		- Пользователь ставит книге статус (прочитал/хочу/читаю).
		- Пользователь удаляет книге статус (прочитал/хочу/читаю).
		- Пользователь обновляет книге статус (прочитал/хочу/читаю).
		- Пользователь оставляет заметку.
		- Пользователь удаляет заметку.
		- Пользователь изменяет заметку.
		- Пользователь оставляет комментарий.
		- Пользователь изменяет комментарий.
		- Пользователь удаляет комментарий.

Работа со списками
		- Пользователь создает персональный список книг.
		- Пользователь удаляет персональный список книг.
		- Пользователь изменяет видимость персонального списка книг (публичный/приватный).
		- Пользователь добавляет книгу в персональный список ("100 книг к прочтению", "Американская фантастика" и т.п.).
		- Пользователь удаляет книгу в списке.
		- Пользователь задает настройки приватности списку (публичный/приватный). По умолчанию - публичный .

Работа с пользователями
		- Пользователь подписывается на комментарии другого пользователя.
